# E-Commerce Sales Performance Dashboard

## Project Description

This project presents an interactive Power BI dashboard built using the Superstore dataset. The dashboard helps analyze sales performance, profitability, regional trends, and product category performance through various visualizations and filters.

## Problem Statement

Businesses need to monitor sales and profit trends to make informed decisions. This project aims to identify key sales patterns, profitable regions, and high-performing product categories using historical sales data.

## Data Source

* Dataset: Superstore Sales Dataset
* Format: CSV
* Records: 9,994
* Features: Sales, Profit, Orders, Categories, Regions, Customers, Discounts, and more.

## Methodology

1. Imported the dataset into Power BI.
2. Checked for missing values and duplicates.
3. Verified data types and data quality.
4. Created KPI cards for Sales, Profit, Orders, and Average Sales.
5. Developed interactive visualizations and slicers.
6. Analyzed trends and generated business insights.

## Dashboard Features

* Total Sales
* Total Profit
* Total Orders
* Average Sales
* Monthly Sales Trend
* Sales by Category
* Sales by Region
* Sales by Sub-Category
* Profit by Region
* Interactive Filters (Region, Category, Segment)

## Key Findings

* Technology and Office Supplies contributed significantly to sales.
* Regional sales performance varied across different regions.
* Monthly sales trends highlighted seasonal fluctuations.
* Certain sub-categories generated higher revenue than others.
* Profitability differed across regions and product groups.

## Results

The dashboard provides an interactive view of business performance, allowing users to explore sales and profit trends through dynamic filtering and visual analysis.

## Conclusion

This project demonstrates how Power BI can transform raw sales data into actionable business insights. The dashboard supports data-driven decision-making through clear visualizations and interactive exploration.
